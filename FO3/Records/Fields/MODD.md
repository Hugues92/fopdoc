MODD Field
==========

As used in the [ACTI](../ACTI.md) and [ADDN](../ADDN.md) record types.

## Format

Count | Name | Type | Info
------|------|------|-----
- | FaceGen Model Flags | uint8 | See below for flag values.

### Flag Values

Flag | Meaning
-----|--------
0x00000001 | Head
0x00000002 | Torso
0x00000004 | Right Hand
0x00000008 | Left Hand
